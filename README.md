# Image Tailor - Download Website

Distribute the Image Tailor Chrome extension to the first 20 people.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Add your Image Tailor extension as `extension.zip` in this folder

3. Start the server:
   ```bash
   npm start
   ```

4. Open http://localhost:3000 in your browser

## Features
- Download counter (0/20)
- Automatic limit enforcement
- Download tracking persists across server restarts
- Clean, responsive UI

## Reset Downloads
Delete `download-count.txt` to reset the counter.
